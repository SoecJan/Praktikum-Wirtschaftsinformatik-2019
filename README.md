# Praktikum-Wirtschaftsinformatik-2019: FAJKER

Zeiterfassung leicht gemacht.

### Installation

Stückliste
- 1x Arduino Nano
- 3x 220 Ohm Widerstand
- 1x Potentiometer
- 1x 16x2 Display
- 1x Analog Stick
- 1x RFID-RC522
- 1x Piezo-Buzzer
- 2x LEDs
- Jumper Wire & Bread Board
